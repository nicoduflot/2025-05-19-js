DOM Introduction

Document Object Model

Tous les éléments de la page sont des objets, et ils ont tous la même structure : mêmes attributs et mêmes méthodes, la différence est dans les valeurs contenues ou non de ces attributs.

Il faut, pour repérer les éléments avec javascript, que toutes la page soit chargée

Quand une page web a fini de cherger tous ses éléments, la fenêtre où apparaît le document envoit l'événement "DOMContentLoaded". En surveillant cet événement, on est capable de faire lire les scripts JS après le chargement complet de la page
